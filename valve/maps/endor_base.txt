INSTALLATION:

1. 	Run Winrar and unzip endor_base.zip

2. 	Open the endor_base MAPS folder and place the endor_base.bsp, endor_base.res, and endor_base.txt in your valve/MAPS folder.  

3.	Open the endor_base GFX folder.  Inside is an ENV folder containing tga files which make up the map sky.  Place the 6 (six) grimmnight.tga files into your valve/gfx/ENV folder.

4.	Open the endor_base SOUND folder.  Inside is an ATAT folder containing map sounds.  Put the ATAT sounds folder in the valve/SOUND folder.

5.	Open the endor_base SPRITES folder.  Inside is an ATAT folder containing map sprites.  Put the ATAT sprites folder in the valve/SPRITES folder.
